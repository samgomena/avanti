Intro

Touche is a fully responsive HTML5 template ideal for restaurants, cafes, bakeries, bars and other food related websites. Built with Bootstrap v4, it looks perfect on all major browsers, tablets, and phones. Touche is packed with everything you need to create a new website from scratch: a beautiful menu with unlimited categories, an image gallery, functional reservation and contact forms, a newsletter form powered by Mailchimp, and many more. Feel free to use any of the pre-packed pages or create an unlimited number of new ones with more than a dozen of predefined sections.

Key features

  • Functional reservation form
  • Functional contact form
  • Functional newsletter form powered by MailChimp
  • Fully responsive layout
  • Parallax background
  • Powered by Bootstrap v4

    • assets/
      • bootstrap/ - original Bootstrap files
      • css/ - compiled CSS files
      • ico/ - favicon files
      • img/ - images
      • js/ - JavaScript files
      • plugins/ - third-party plugins
      • scss/ - source SCSS files

SASS file compilation

Touche makes use of the Gulp toolkit to handle SASS file compilation and minification. Please follow the steps below to install required dependencies and start compiling SASS files:

  1. Open the command line, navigate to the theme folder and run npm install to install the required dependencies.

  2. Enter gulp to start watching for changes in SASS files and automatically compile them. This will also open the http:localhost:3000 page to preview the changes.

  3. Enter gulp build to compile the SASS files, minify CSS and JS files and images.

Components

Touche includes multiple custom components as well as customized Bootstrap components you can use to create new pages or modify existing ones.

Section

Each page of the theme is built with sections that help you organize and visually separate blocks of content within the same page. 

<section class="section"> ... </section>

with the gray background: 

<section class="section section_gray"> ... </section>

with the top border: 

<section class="section section_border_top"> ... </section>

with the bottom border: 

<section class="section section_border_bottom"> ... </section>

with the top padding set to 0: 

<section class="section section_no-padding_top"> ... </section>

with the bottom padding set to 0: 

<section class="section section_no-padding_bottom"> ... </section>

with the parallax background (requires the Parallax plugin): 

<section class="section" data-parallax="scroll" data-image-src="path-to-image"> ... </section>

In order to make the form work, you need to find, modify and fill in an action URL of your Mailchimp newsletter campaign.

  1. Please follow the instructions here to find an action URL of your newsletter campaign. It should look something like http://xxxxx.us#.list-manage1.com/subscribe/post?u=xxxxx&id=xxxx.

  2. Replace post?u= with post-json?u=.

  3. Add &c=? at the end. Your string should now look something like http://xxxxx.us#.list-manage1.com/subscribe/post-json?u=xxxxx&id=xxxx&c=?.

  4. Replace the value of the action attribute in the form code above with the results: 

form class="section_newsletter__form validate" action="http://xxxxx.us#.list-manage1.com/subscribe/post-json?u=xxxxx&id=xxxx&c=?" method="get" id="mc-embedded-subscribe-form" name="mc-embedded-subscribe-form" target="_blank" novalidate=""> ... </form>

Mailchimp signup forms have a unique ID, so it's not possible to have more than one form on a single page. Instead, use an example below to add a clone: 

<form class="section_newsletter__form section_newsletter__form_clone">
	<div class="row">
		<div class="form-group col-md-9">
			<label class="sr-only">E-mail address</label>
			<input type="email" class="form-control" placeholder="Email address">
		</div>
		<div class="col-md-3 text-center">
			<button type="submit" class="btn btn-primary">
				Subscribe
			</button>
		</div>
	</div>
</form>

This form cannot be used alone and needs an original signup form on the same page to submit new signups.

Parallax plugin

Touche makes use of the Parallax.js plugin to handle the parallax effect in the header section of every page.

To easily add a parallax effect behind an element, add data-parallax="scroll" to the element you want to use, and specify an image with data-image-src="/path/to/image.jpg". 

<div class="parallax-window" data-parallax="scroll" data-image-src="/path/to/image.jpg"></div>

Please refer to the official documentation for more info.

Flickity plugin

Touche makes use of the Flickity plugin to create sliders (carousels) in the following sections:

  • Featured dishes
  • Testimonials
  • Gallery

Flickity works on a container carousel element with a group of cell elements. 

<div class="main-carousel">
  <div class="carousel-cell">...</div>
  <div class="carousel-cell">...</div>
  <div class="carousel-cell">...</div>
  ...
</div>

Flickity can be initialized with jQuery, vanilla JS, and HTML. In Touche we initialize Flickity with jQuery in /assets/js/theme.js(theme.min.js).

Please refer to the official documentation for more info.

Lightbox plugin

Touche makes use of the Lightbox plugin to handle modal windows with images in the Gallery section.

Initialize with HTML:

Single images. Add a data-lightbox attribute to any image link to enable Lightbox. For the value of the attribute, use a unique name for each image. For example: 

<a href="images/image-1.jpg" data-lightbox="image-1" data-title="My caption">Image #1</a>

Image sets. If you have a group of related images that you would like to combine into a set, use the same data-lightbox attribute value for all of the images. For example: 

<a href="images/image-2.jpg" data-lightbox="roadtrip">Image #2</a>
<a href="images/image-3.jpg" data-lightbox="roadtrip">Image #3</a>
<a href="images/image-4.jpg" data-lightbox="roadtrip">Image #4</a>

Please refer to the official documentation for more info.

Isotope plugin

Touche makes use of the Isotope plugin to handle layout and filtering of elements in the following sections:

  • Menu
  • Gallery

Isotope works on a container element with a group of similar child items. 

<div class="grid">
  <div class="grid-item">...</div>
  <div class="grid-item grid-item--width2">...</div>
  <div class="grid-item">...</div>
  ...
</div>

In Touche we initialize Isotope with jQuery in /assets/js/theme.js(theme.min.js).

Please refer to the official documentation for more info.

Alerts plugin

Touche makes use of the custom Alerts plugin to handle alerts in the following forms:

  • Contact form
  • Reservation form
  • Newsletter signup form

Syntax: 

$(document).trigger('touche.alert.show', [type, message]);

The following options are available:

type (string): one of the four available types:

  • success
  • danger
  • warning
  • info

message (string): message to appear in the alert.

For example: 

$(document).trigger('touche.alert.show', ['success', 'A new alert has been triggered successfully!']);

Google maps

Touche makes use of the Google maps to show a map in the Map section. Map coordinates can be controlled via HTML with the following attributes:

  • data-lat (float) - latitude.
  • data-lng (float) - longitude.
  • data-zoom (number) - zoom level (0 - 19).
  • data-info (string) - info window content (optional).

Please refer to the official documentation for more info.

FontAwesome

Touche makes use of the FontAwesome library to show icons. Use the following syntax to add a new icon on a page:

Please refer to the official documentation for the list of selectors available and a more detailed info.
